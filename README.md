# Nova Digital Website

Sito web moderno e responsive per Nova Digital - Trasformiamo le tue idee in risultati.

## 🎨 Caratteristiche

- **Design Moderno**: Colori blu, bianco e violaceo
- **Responsive**: Perfetto su desktop, tablet e mobile
- **Smooth Scrolling**: Navigazione fluida tra le sezioni
- **Animazioni**: Effetti al caricamento e all'hover
- **SEO Friendly**: Struttura HTML semantica

## 📁 Struttura dei File

```
nova-digital-website/
├── index.html       # Pagina principale
├── styles.css       # Stili e design
├── script.js        # Interattività
└── README.md        # Documentazione
```

## 🚀 Sezioni del Sito

1. **Header**: Logo e navigazione
2. **Hero Section**: Titolo principale e CTA
3. **Servizi**: 4 categorie di servizi
4. **Perché scegliere Nova Digital**: 5 punti di forza
5. **Contatti**: Link a Instagram e TikTok
6. **Footer**: Informazioni di copyright

## 🎨 Colori Utilizzati

- Blu Primario: `#0066cc`
- Violetto Secondario: `#7c3aed`
- Sfondo Chiaro: `#f8f9ff`
- Bianco: `#ffffff`

## 📱 Responsive Design

- Desktop: Layout completo
- Tablet: Griglia adattiva
- Mobile: Stack verticale ottimizzato

## 💡 Come Personalizzare

1. Modifica il contenuto in `index.html`
2. Personalizza i colori in `styles.css` (variabili CSS)
3. Aggiungi funzionalità in `script.js`
